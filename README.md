## フォルダの構成
### ファイルの種類
HP作成には基本的にHTML, CSS, JavaScriptの3種類のファイルと、画像などの添付ファイルが必要で、大まかな使い分けは以下の通り。

- `html`: ホームページに記載する内容を記述するファイル。ページごとに1つのファイルを作成し、互いにリンクで結びつける。
- `css`: ホームページの見た目を整えるファイル。ページは変わっても体裁は同じ方が良いことが多いので、基本は共通のファイル (今回は`styles.css`) を作成して使いまわす。
- `JavaScript`: ホームページの込み入った動作を指定するプログラムファイル。リンクやページ遷移程度であれば`html`でも記述できるが、タイマーとかを実装するには必要。こちらも1つのファイル (今回は`menu.js`) を作成し、使いまわすことが多い。

### その他フォルダ
基本はフォルダ名の通り。`images`はトップページに使用している画像を、`blog_image`はブログで使っている画像の置き場。
